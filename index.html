Clovrr

A blogging app

Explaining.

Blogs I've had in the past were great, but not quite what I wanted.

I wanted to be able to easily get all my posts out (if needed). I wanted to have Markdown support. I wanted a powerful templating system.

Nothing out there fulfilled these requirements, so I set out to make my own.

Here's version 1. It's rough, but it's here. I spent about 2 weeks working on it.

Downloading.

Download v1.0 (.zip) or Download the in-development version.

Also, you can check out the GitHub repository to view the in-dev code.

Requirements.

  • Ruby MRI 1.9.2 or its equivalent
  • A server that can serve Rails apps
  • MongoDB
  • A brain

Installing.

Clovrr was sort of designed to work on Heroku, but it could be installed most anywhere Rails 3 apps work otherwise.

Anyway, here's the Heroku Way.

  1. Download Clovrr via the links above or clone the git repo
    (git clone git://github.com/evanwalsh/clovrr.git)
  2. Install the Heroku gem (gem install heroku)
  3. Make a new app on Heroku: heroku create YOUR_APP_NAME --stack bamboo-mri-1.9.2
  4. Modify config/mongoid.yml if necessary (Clovrr will automatically work with Heroku and MongoHQ)
  5. (OPTIONAL) Modify db/seeds.rb
  6. git push heroku master
  7. heroku rake app:install
